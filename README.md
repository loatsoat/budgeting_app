# template_student_repo



## Instructions

This project template contains five folders (`ms1` to `ms5`) that correspond to each milestone of the semester project.

Store the current version of your project documentation (aka *Projektmappe*) as PDF-file in the respective folder **BEFORE THE MILESTONE'S DEADLINE**. If you miss storing the file in time, it will be graded with **0 points**!

Use the `src` folder for the development of your app and the `doc` folder for the collaborative and continuous editing of your project documentation (aka *Projektmappe*).
